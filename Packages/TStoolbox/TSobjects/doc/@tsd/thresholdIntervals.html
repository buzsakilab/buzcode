thresholdIntervals

PURPOSE

Returns intervals in which a TSD is above (below) threshold

SYNOPSIS

function is = thresholdIntervals(tsa, thr, varargin)

DESCRIPTION 

  Returns intervals in which a TSD is above (below) threshold
  
      USAGE:
      is = thresholdIntervals(tsa, thr, OptionName, OptionValue)
      
      INPUTS:
      tsa - a tsd object
