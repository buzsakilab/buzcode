SYNOPSIS

function addframe(vw,img)

DESCRIPTION 

WORKED=ADDFRAME(VW)
  Writes a video frame.  VW must be a videoWriter object and IMG a 2D
  numeric array with 1 or 3 channels that is autoconverted to a uint8
  image as follows: 

    type    assumed range
    ----    -------------
    uint8   0 to 255
    double  0 to 1
    logical 0 to 1
